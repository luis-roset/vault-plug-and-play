## 1.0.0 (July 16, 2020)

Hashicorp Vault Enterprise users can take advantage of this
Splunk® app to understand Vault from an operational and security
perspective. The Splunk app includes powerful dashboards that split
metrics into logical groupings targeting both operators and security
teams. The app can be used with Splunk Enterprise as well as
Splunk Cloud.

Features:
  * Summary dashboard
  * Operational metrics dashboard
  * Usage metrics dashboard
  * Audit log dashboard
  * Storage dashboard
  * Quota monitoring dashboard
  * Identify sources of high-TTL tokens
